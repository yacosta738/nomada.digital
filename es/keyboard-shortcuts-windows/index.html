Keyboard Shortcuts for Windows

Window Management

  1. Win
    E
    X
    C
    Enter
    Start Microsoft Excel by searching for it. Of course this works for all apps and also for finding files.
  2. Win
    +
    E
    Open Windows Explorer
  3. Win
    +
    D
    Show the Desktop
  4. Win
    +
    L
    Lock the computer
  5. Alt
    +
    Tab
    Switch app
  6. Win
    +
    ←
    or
    →
    Place window on the left or right half of the display
  7. Win
    +
    ↑
    or
    ↓
    Maximise or minimise a window
  8. Win
    +
    Shift
    +
    ↑
    Make a windows full height
  9. Win
    +
    P
    Use an external display or projector
  10. Ctrl
    +
    N
    Open new window (of current app)
  11. Shift
    +
    F10
    Same as right click

Shortcuts for Web Browsers

Seems to work in all browsers, but it's not guaranteed.

Common

  1. Ctrl
    +
    Mouse Cursor
    Open clicked link in new tab
  2. Ctrl
    +
    F
    Search on the current page
  3. F3
    Go to the next result in a search
  4. Ctrl
    +
    T
    Open in new tab
  5. Ctrl
    +
    Shift
    +
    T
    Open last closed tab(s)
  6. Ctrl
    +
    W
    Close current tab
  7. F11
    Full screen mode
  8. F5
    Reload the page
  9. Ctrl
    +
    F5
    Reload the page without cache, i.e. if the page isn't working properly
  10. Ctrl
    +
    L
    Focus the address bar
  11. Ctrl
    +
    J
    Show downloaded files
  12. Ctrl
    +
    +
    Zoom in
  13. Ctrl
    +
    -
    Zoom ut
  14. Ctrl
    +
    0
    Zooma till 100%
  15. Ctrl
    +
    Mouse wheel
    Zoom in or out. Works in Word and some other apps as well.
